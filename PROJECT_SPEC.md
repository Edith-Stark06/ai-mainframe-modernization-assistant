# AI-Powered Mainframe Modernization Assistant

## Vision

Create an enterprise platform capable of understanding legacy IBM Z applications and assisting organizations with modernization.

The system will combine deterministic static analysis with Large Language Models.

## Core Modules

- FastAPI Backend
- Streamlit Dashboard
- File Ingestion
- COBOL Parser
- JCL Parser
- Copybook Parser
- Static Analysis
- Business Rule Extraction
- Documentation Generator
- AI Assistant
- RAG Engine
- Modernization Advisor
