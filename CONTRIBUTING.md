## Commit Format

feat:

## Pull Request Checklist

- Architecture followed
- Tests passing
- Ruff clean
- Black formatted
- MyPy clean
- Documentation updated

---

## Code Style

- Type hints required
- Docstrings required
- No print()
- Absolute imports only
